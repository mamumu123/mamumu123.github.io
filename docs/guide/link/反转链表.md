## 题目描述

输入一个链表，反转链表后，输出新链表的表头。


``` js
function ReverseList(pHead)
{
    // write code here
    if(!pHead){return null}
    let cur = null
    
    while(pHead){
        let tem = pHead.next;
        pHead.next = cur
        cur = pHead
        pHead = tem
        
        // [curr.next,prev,curr]=[prev,curr,curr.next]
    }
    return cur
}
```
